# 第3章：天空被撕裂的那一天（下）

楚锐跟着白袍人，走向祭坛。

他的每一步都很小心，每一步都在计算着退路。如果被发现，他能从这个地方逃出去吗？这里有二三十个信徒，还有那个金袍人，他身上的能量波动不弱。

【评估：如果暴露，生存概率低于40%。】

"闭嘴。"楚锐在心里说，"我知道。"

白袍人在金袍人面前停下，低声说了几句。金袍人转过头，看向楚锐。那双眼睛很锐利，像是要看穿他的灵魂。

"你说你妹妹被带走了？"金袍人问。

"是的。"楚锐说，"她叫楚瑶。三天前，被采收舰带走了。"

"三天前..."金袍人若有所思，"第7分区的全面收割。我知道那次行动。"

他走近了一步，仔细地打量着楚锐。

"你想知道她的下落？"

"我想知道她是不是还活着。"

金袍人笑了，那种笑容里有一种...怜悯？"活着？当然还活着。纯净脑域是珍贵的资源，他们不会浪费。"

"资源..."楚锐握紧了拳头，"你们把人类当成资源？"

"不。"金袍人摇头，"我们是在帮助他们升华。肉体的痛苦是暂时的，灵魂的永恒才是真实的。"

他说着，指向那个屏幕。

"你想看看她吗？"

楚锐的心跳加速了。"我可以？"

"可以。"金袍人说，"但你要明白，一旦看到了，你就无法回头了。你会知道真相，而真相...往往是残酷的。"

"让我看。"

金袍人走到控制台前，按下了几个按钮。屏幕上的画面变了，不再是那些模糊的容器，而是一个特写——

楚瑶的脸。

她躺在那里，眼睛闭着，脸色苍白，但还在呼吸。她的身体被各种管子连接着，有一些液体在管子里流动，发出微弱的光芒。

【生命体征：稳定，但微弱。】引擎说，【她正在被...养护。他们在保持她的生命，等待提取的最佳时机。】

"最佳时机？"楚锐在心里问。

【纯净脑域的提取需要特定的条件。必须是脑域开发度达到顶峰的时候，提取的能量才是最大的。他们可能在用某种方法刺激她的脑域开发。】

楚锐看着屏幕上的妹妹，心中的怒火在燃烧。

"她...痛苦吗？"他问金袍人。

"不。"金袍人说，"她处于深度休眠状态，感觉不到任何痛苦。当她醒来的时候，她就会成为永恒的一部分。"

"谎言。"楚锐说，声音很低，但充满了愤怒。

金袍人转过头，看着他。那双眼睛里闪过一丝...警觉？

"你说什么？"

"我说你在说谎。"楚锐抬起头，直视着他的眼睛，"我知道会发生什么。她的灵魂会被抽干，她的身体会变成空壳。没有什么永恒，只有死亡。"

金袍人的脸色变了。他后退了一步，手伸向腰间的某个装置。

"你是谁？"他问，声音变得冰冷，"你不是普通的拾荒者。"

"我是她哥哥。"楚锐说，"我是来救她的人。"

他动了。

速度比金袍人想象的要快得多。楚锐一拳砸向控制台，那个屏幕在火花中炸裂。然后他转身，向教堂的大门冲去。

"拦住他！"金袍人大喊，"他是亵渎者！"

那些信徒反应过来了，他们从长椅上跳起来，向楚锐扑来。但楚锐更快，他开启了【肉体重构】，身体表面覆盖上一层淡金色的光芒，直接撞开了两个挡路的信徒。

【声波感知】全力运转，他在脑海中"看"到了整个教堂的布局，看到了每一个敌人的位置，看到了——

那个金袍人举起了手，手中有一个小小的装置，正在发光。

【警告：检测到能量武器充能。】

楚锐向侧面翻滚，一道蓝色的光束从他身边擦过，在墙上烧出一个大洞。

他不能在这里战斗。人太多，地方太狭窄。

他冲向窗户，撞碎了彩色玻璃，跳了出去。

外面是黑夜。他落在地上，翻滚卸力，然后爬起来就跑。身后传来喊叫声，有更多的信徒追了出来。

但楚锐的速度更快。他在废墟中穿梭，利用【声波感知】避开障碍物，很快就甩掉了追兵。

当他终于停下来的时候，他已经跑出了几公里，肺在燃烧，腿在颤抖，但他还活着。

他靠在一堵墙上，大口大口地喘气。

他看到了楚瑶。她还活着，虽然被关在那种容器里，但她还活着。

这就够了。

"我会救你出来的。"他对着夜空说，"不管用什么方法。我会变得足够强，强到能撕裂那些飞船，强到能把那些杂种一个个撕碎。"

"等我，楚瑶。"

"等我。"
