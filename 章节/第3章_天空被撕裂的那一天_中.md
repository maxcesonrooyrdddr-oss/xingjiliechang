# 第3章：天空被撕裂的那一天（中）

教堂里面的景象和外面截然不同。

如果说外面是废土的破败，那么里面就是...某种扭曲的圣洁。墙壁上挂满了白色的帷幔，地板上铺着干净的地毯，空气中弥漫着一种甜腻的香气。

教堂里有不少人。大约二三十个，都穿着白色的长袍，坐在长椅上，低着头。最前方有一个祭坛，上面有一个巨大的屏幕，闪烁着蓝色的光芒。屏幕上显示着一个巨大的六边形，里面有一只睁开的眼睛。

"坐下吧。"守卫说，"仪式马上就要开始了。"

楚锐找了个角落的位置坐下，观察着周围。这些人看起来都很...正常。不是那种疯狂的邪教徒，而是普通的拾荒者。他们的脸上带着一种...期待？

【检测到高强度能量波动。】

【来源：前方屏幕。】

【警告：那是一种精神控制装置。】

"精神控制？"

【低级别的。只能影响意志薄弱的人。以你现在的精神强度，它只能让你感到不适。】

楚锐点点头，继续保持警惕。

然后，仪式开始了。

一个穿着金色长袍的人从祭坛后面走出来。他的胸前也有那个眼睛的标志，但是金色的。

"兄弟姐妹们。"他的声音很洪亮，"欢迎来到今天的升华仪式。"

人群安静下来。

"我们知道，这个世界是痛苦的。"金袍人说，"但我们要相信，这一切都有意义。我们的主人，那些高高在上的存在，他们不是恶魔，他们是救世主！"

他张开双臂。

"他们用他们的技术，向我们展示了一条通往永恒的道路！肉体是软弱的，但灵魂——灵魂是永恒的！"

人群开始骚动，有人开始哭泣，有人举起手。

楚锐看着这一切，感到一种...恶心。他知道这种话术——用希望来包装绝望。

但今天，他不是来揭穿他们的。他是来寻找信息的。

仪式继续进行。金袍人开始念诵某种祷词，用的是外星语。人群跟着念诵。

然后，到了仪式的"高潮"。

金袍人按下了一个按钮，屏幕上显示出了一个新的画面。那是...一艘采收舰的内部？楚锐看到了那些透明的容器，里面装着人，他们的身体上连着各种管子。

"看！"金袍人激动地喊道，"这就是升华！这些人，他们正在摆脱肉体的束缚，他们的灵魂正在被净化！"

楚锐的眼睛死死地盯着屏幕。他在那些容器里寻找着。

然后，他看到了她。

楚瑶。

她躺在其中一个容器里，闭着眼睛，像是睡着了。她的身体上连着那些管子，但她的表情...很平静。

楚锐感觉自己的心脏停跳了一拍。

她还活着。至少，她的身体还活着。

"引擎。"他在心里喊道，"你能感觉到她吗？"

【距离太远。】引擎说，【但那确实是她。她的生命体征...很微弱，但还存在。】

"怎么救她？"

【那个屏幕。它连接着采收舰的监控系统。如果你能接触到那个控制台，也许我能入侵他们的系统。】

"我需要接近那个控制台。"

【风险极高。】

"我知道。"

楚锐深吸一口气，开始思考计划。

仪式结束了，人群开始散去。有些人走向祭坛，向金袍人询问什么。楚锐混在人群中，慢慢地向前移动。

然后他听到了一段对话。

"大人，"一个中年女人问道，"我的儿子...他有纯净脑域的潜质吗？"

金袍人看着她，那种眼神让楚锐感到不寒而栗——那是评估货物价值的冷漠。

"也许。"他说，"我们需要测试。"

女人拉着一个大约十岁的男孩，走向教堂的后面。

楚锐跟了上去。

教堂后面有一个小房间，像是一个诊所。金袍人拿起一个头盔一样的东西，戴在男孩头上。那个头盔开始发光。

【那是脑域扫描仪。】引擎说，【可以测量脑域开发度。】

几秒钟后，扫描结束了。金袍人看着仪器，摇了摇头。

"很遗憾。只有4.2%。不够纯净。"

女人的脸一下子白了。

"但别担心。"金袍人说，"我们可以尝试一些...辅助手段。一种药物，可以刺激脑域开发。但有副作用——可能会损伤大脑。"

女人犹豫了。她看着自己的儿子。

"我...我需要考虑一下。"

"当然。"金袍人微笑着，"你可以慢慢考虑。但记住，机会不等人。"

楚锐躲在门外，听着这一切。他的拳头握得紧紧的。

这些人...他们不只是信徒，他们是帮凶。

【有人发现你了。】

引擎的警告来得太迟。楚锐感觉到一只手拍在他的肩膀上，他转身，看到一个穿着白袍的人，正用怀疑的眼神看着他。

"你在这里干什么？"那个人问，"后面是禁区，外人不能进入。"

楚锐没有回答。他的大脑在飞速运转。

"我...我想了解更多关于升华的事。"他说，"我有妹妹，她被带走了。我想知道...她是不是也在那里。"

那个人的眼神软化了一点。"你妹妹？"

"是的。她叫楚瑶。她被采收舰带走了。"

白袍人沉默了。然后他叹了口气，"跟我来。也许...我可以帮你问问。"

他转身，走向金袍人。楚锐跟在后面，心跳得像是要从胸口跳出来。

他不知道这是机会还是陷阱。但他已经没有退路了。
