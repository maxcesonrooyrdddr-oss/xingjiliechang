# 第3章：天空被撕裂的那一天（中）

楚锐在聚集地的边缘找到了一个废弃的仓库，暂时住了下来。

他需要时间来消化那本笔记本里的信息，也需要时间来计划下一步。盲目的狩猎只会让他死得更快，他需要策略。

【建议：寻找其他觉醒者。】引擎说，【在这个星球上，不止你一个拥有远古基因序列。其他碎片宿主可能也在某个地方觉醒。】

"其他碎片？"

【真理引擎破碎成了七块。你拥有的是最大的一块，但其他六块也散落在地球上，藏在某些人类的基因里。】

"他们能帮我？"

【或者杀了你。】引擎说，【碎片之间会相互吸引，也会相互排斥。当两个宿主相遇时，他们的本能会驱使他们吞噬对方。】

"同类相食？"

【进化从来都是残酷的。】

楚锐沉默了一会儿。然后他问道："你能感觉到其他碎片吗？"

【距离太远。】引擎说，【但如果你接近到10公里以内，我能给出大致的方向。】

"10公里..."楚锐苦笑，"在这个废土上，10公里可能意味着生死。

他开始整理笔记本里的信息。陈明——那个科学家——在笔记本里记录了很多关于外星人的观察。其中有一段特别引起了楚锐的注意：

"采收舰并不是无敌的。它们的护盾对物理攻击有很强的防御力，但对某种特定频率的能量波动很脆弱。这种频率...很奇怪，像是一种共鸣，一种与宇宙基本法则的共鸣。"

"引擎，这是什么意思？"

【他在说"真理之力"。】引擎说，【这是我原本的力量，一种能够干涉现实法则的能量。现在的我虽然破碎了，但如果你进化到足够的程度，你就能重新唤醒这种力量。】

"足够高的程度是多少？"

【超脱境。】引擎说，【第九境界，最后一个境界。到了那个程度，你可以扭曲物理法则，可以撕裂空间，可以...做任何你想做的事。】

"包括救回楚瑶？"

【包括让死者复生。】

楚锐握紧了拳头。那是一个遥远的目标，但至少是一个目标。他不是漫无目的地在变强，他是在走向一个明确的终点。

他继续往下读。然后，他看到了一张地图。

那是陈明手绘的地图，标注着这个区域的所有重要地点。其中一个标记引起了他的注意——一个画着红叉的地方，旁边写着："降临派据点，极度危险"。

"降临派？"楚锐皱起眉头。

他翻了几页，找到了关于降临派的描述：

"有一群人，他们相信外星人是来拯救人类的。他们认为人类的肉体是软弱的，只有通过外星人的'升华'，人类才能获得真正的进化。他们自愿成为外星人的仆从，帮助他们管理牧场，甚至献祭自己的同胞。"

楚锐的眼神变得冰冷。他见过这种人在废土上并不少见。绝望会让人做出疯狂的事。

"笔记本里有没有说怎么找到他们？"

【你为什么要找他们？】引擎问。

"因为他们可能有信息。"楚锐说，"关于采收舰的，关于纯净脑域的。如果他们真的为外星人工作，他们一定知道怎么进入那些飞船。"

【风险极高。】

"我知道。"

【但你还是要去。】

"我必须去。"楚锐说，"为了救回楚瑶，我愿意冒任何风险。"

他站起来，把笔记本收好，走出仓库。

地图显示，那个降临派据点在废墟的东边，大约十五公里的路程。中间要经过一片高辐射区，还要绕过两个掠夺者帮派的地盘。

他检查了一下装备。一把刀，几块变异兽的骨头作为货币，还有一点干净水。不够，但也没办法。在废土上，你永远不会有足够的装备。

他出发了。

高辐射区的路比他想像的还要难走。空气中有一种刺鼻的气味，让眼睛和喉咙都疼。他开启了【肉体重构】，让身体细胞加速分裂和死亡，以此来抵抗辐射的侵蚀。但这个过程很痛苦，像是全身都在被火烧。

【警告：辐射水平超出安全阈值。】

【建议：快速通过，不要停留。】

楚锐加快了脚步。他几乎是跑过那片区域的，肺在燃烧，腿在颤抖，但他没有停下来。

当他终于走出辐射区的时候，他吐了。吐出来的东西里有血，还有某种黑色的、像是焦油一样的液体。他的身体在自我修复，但那种修复的代价很高。

他休息了几分钟，然后继续前进。

掠夺者帮派的地盘比辐射区更危险。那些人不只是想要你的东西，他们想要你的命。他们在废土上游荡，像是一群饥饿的狼，寻找着落单的猎物。

楚锐听到了他们的声音——在左边，大约两百米外。有笑声，还有金属碰撞的声音。他在引擎的指引下，绕了一个大圈，避开了他们的巡逻路线。

【前方：降临派据点。】

楚锐停下来，躲在一堵倒塌的墙后面，观察前方。

那是一座教堂。不是那种宏伟的教堂，是一座小型的社区教堂，但在废土上，它保存得异常完好。墙壁被重新粉刷过，窗户上的彩色玻璃完整无损，门口甚至还有两盆花——真正的花，在这个铁锈色的世界里显得格外诡异。

门口有两个守卫，穿着白色的长袍，胸前绣着那个标志——六边形里的眼睛。外星人的标志。

楚锐观察了一会儿。守卫看起来很放松，不像是在防备什么。也许是因为他们相信外星人会保护他们，也许是因为在这个区域，没人敢惹降临派。

他决定直接进去。

他从阴影中走出来，向教堂走去。守卫发现了他，举起手中的武器——不是冷兵器，是那种能量武器，从外星人那里获得的。

"站住。"其中一个说，"你是谁？来干什么？"

"我是一个寻求者。"楚锐说，他早就想好了说辞，"我想要...升华。"

守卫对视了一眼。然后其中一个笑了，那种笑容里有一种...狂热。

"又一个迷途的羔羊。"他说，"进来吧。兄弟会欢迎你。"

楚锐走进教堂，心跳加速，但他的脸上保持着平静。

他不知道里面有什么在等着他。但他准备好了。
