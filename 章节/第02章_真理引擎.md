# 第2章：真理引擎

**降临日后第7年，辐射纪元，第3天。**

**第9号生物质牧场，废土边缘。**

楚锐站在一座倒塌的高架桥上，俯视着下方那片被辐射污染的沼泽地。

在他身下三十米处，一只变异巨鼠正在啃食一具人类的残骸。那只巨鼠的体型如同一头成年公牛，灰色的皮毛上长满了拳头大小的瘤子，每一次呼吸都喷吐着带着辐射尘的绿色雾气。

在七年前，这种怪物是废土上的霸主，是人类闻风丧胆的死神。它们的爪子可以轻易撕裂钢铁，它们的牙齿可以咬碎混凝土，它们的速度可以追上全速行驶的摩托车。

无数拾荒者死在它们的口中，成为它们的食物和养料。

但现在——

楚锐的瞳孔中闪过一丝淡金色的光芒。

【目标：变异巨鼠（D级辐射变异兽）】
【威胁等级：低】
【特殊能力：剧毒撕咬、辐射喷吐】
【核心弱点：颈部脊椎第三节，神经中枢节点】
【建议战术：正面突击，利用速度优势斩断颈椎】

【当前状态：凡星境 Level 0（引擎损伤87%，急需修复）】

【吞噬此目标可恢复：3%引擎损伤】

信息直接涌入了楚锐的大脑，不是学习，不是理解，而是……**下载**。就像有人把这些知识直接拷贝进了他的记忆库，让他瞬间就知道了该如何杀死下方那头怪物。

这就是"零号真理引擎"的力量。

三天前，当楚锐亲眼目睹妹妹楚瑶被采收舰抓走时，他爆发出了令灵魂泣血的极致悲痛，瞬间点燃了远古基因序列中沉睡的代码，唤醒了这台传说中的超级计算机碎片。

或者说，是这台超级计算机选择了他。

"我……终于等到你。"

那是引擎苏醒时说的第一句话，虚弱得像是一阵即将消散的烟雾。

"等待了七万年……终于有一个宿主，符合觉醒条件……"

七万年。

楚锐不知道那意味着什么，但他知道一件事——

这台引擎，是他救回妹妹的唯一希望。

"告诉我，"他在心中问道，"怎样才能救回瑶瑶？"

引擎沉默了很长时间，然后给出了答案：

【目标：救出妹妹楚瑶】
【路径：变强 → 猎杀外星人 → 夺取他们的科技 → 修复引擎 → 杀上母舰】
【当前阶段：凡星境 Level 0 → Level 9】
【所需生物质：约1000个D级目标，或100个C级目标，或10个B级目标】

一千个D级目标。

楚锐看向下方那头变异巨鼠，眼中燃烧着冰冷的火焰。

那就从第一个开始。

他深吸一口气，然后——

跳了下去。

三十米的高度，在重力加速度的作用下，楚锐的下坠速度达到了每秒二十五米。他的身体像是一颗黑色的炮弹，直直地砸向那头变异巨鼠。

巨鼠察觉到了危险，它猛地抬起头，血红的双眼锁定了空中的楚锐。它的反应速度快得惊人，在楚锐即将落地的瞬间，它已经做出了规避动作——

但楚锐更快。

【肉体重构（初级）】发动。

楚锐感觉自己的肌肉纤维在瞬间重组，变得更加致密，更加强健。他的骨骼密度提升了三倍，他的反应速度提升了五倍，他的力量——

提升了十倍。

轰！！！

楚锐的双脚狠狠地踩在了巨鼠的背上，巨大的冲击力将巨鼠直接砸进了沼泽地中，溅起了数米高的污泥。

巨鼠发出一声愤怒的嘶吼，它猛地翻身，锋利的爪子朝着楚锐横扫而来。那一爪的速度突破了音速，在空中留下了一道淡淡的白色气浪。

但楚锐已经预判到了。

【基础扫描】发动。

在他的视野中，巨鼠的攻击轨迹被清晰地标记出来，就像是一条红色的箭头。他提前0.3秒做出了规避动作，身体以一种不可思议的角度扭曲，险之又险地避开了那一爪。

然后，他的右手如闪电般探出，五指并拢如刀，狠狠地刺入了巨鼠的颈部——

那个真理引擎标记出来的弱点，神经中枢的节点。

【肉体重构】强化过的指骨轻易地撕裂了巨鼠厚实的皮毛和肌肉，精准地切断了它的颈椎。

巨鼠的身体瞬间僵直，它的眼中闪过一丝难以置信的光芒，然后——

轰然倒地。

【击杀成功。】

【开始吞噬……】

楚锐感觉自己的手掌传来一阵灼热的吸力，那头巨鼠的身体开始以肉眼可见的速度干瘪，某种纯净的能量从巨鼠的体内涌出，顺着楚锐的手臂流入他的身体。

那是"生物质"——引擎修复所需的养料。

【吞噬完成。】

【引擎损伤：87% → 84%】

【获得生物质：微量】

【获得能力碎片：毒素抗性（初级）】

【等级提升：凡星境 Level 0 → Level 1】

一股暖流在楚锐的体内流转，修复着他三天前被采收舰重力场压断的肋骨，强化着他刚刚觉醒的每一个细胞。

他握紧拳头，感受着体内涌动的力量。

这只是第一个。

还有九百九十九个。

"瑶瑶，等着我。"他在心中轻声说道，"哥一定会……把你带回来。"

他弯下腰，从巨鼠的尸体上拔出了自己的手，然后转身，看向废土的深处。

那里有更多的怪物，更多的生物质，更多的……

力量。

楚锐迈出了步伐，向着废土的深处走去。

他的身影在辐射尘中渐渐消失，像是一头刚刚觉醒的猛兽，开始了他的狩猎。

而在他头顶的天空上，那些悬浮的采收塔依然在缓缓运转，像是在俯视着大地上的一切生灵。

它们不知道，一只微不足道的"虫子"，正在悄然崛起。

而总有一天，这只虫子会撕裂这片天空。

**第2章：真理引擎 · 完**
