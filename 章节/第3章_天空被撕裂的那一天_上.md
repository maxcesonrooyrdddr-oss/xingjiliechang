# 第3章：天空被撕裂的那一天（上）

楚锐在废墟中穿行了三天。

三天里，他猎杀了十七只变异兽，从E级的变异老鼠到D级的酸液喷射者。每一次吞噬都让他的力量增长，每一次修复都让他的身体变得更加强韧。

【当前状态：凡星境 Level 5】
【引擎损伤：85% → 72%】
【解锁能力：基础扫描、肉体重构（中级）、声波感知（中级）、骨甲硬化（初级）、毒素抗性（中级）】

他的外表也发生了变化。皮肤变得更加紧致，肌肉线条更加明显，眼睛里偶尔会闪过一丝金色的光芒——那是引擎活跃时的副作用。

但更重要的是他的心态。他不再害怕了。不是因为他变得强大，而是因为他已经没有什么可失去的了。楚瑶已经被带走，他的父母已经死了，他在乎的人都已经不在了。他现在只有一个目标：变强，然后把她救回来。

【前方检测到人类聚集地。】

引擎的声音在他的脑海里响起。楚锐停下脚步，看向前方。那是一片相对完好的建筑群，看起来像是旧时代的商业区。有几栋楼还立着，窗户被木板封住，屋顶上挂着一些布料——那是信号，表示这里有拾荒者居住。

"多少人？"

【约50人。生命体征：多数虚弱，少数健康。】

"有威胁吗？"

【检测到3个武装个体，持有冷兵器。威胁等级：低。】

楚锐思考了一下。他需要信息。关于那些采收舰的，关于纯净脑域的，关于...如何救回楚瑶的。这些人可能知道一些东西。

他走向那片建筑群。

入口处有两个守卫，都是中年男人，手里拿着用钢筋磨成的长矛。他们看到楚锐的时候，眼神里有警惕，但更多的是...疲惫。他们已经见过太多像他这样的拾荒者了。

"来干什么的？"其中一个问。

"换点东西。"楚锐说，"也打听点消息。"

"有货吗？"

楚锐从包里掏出一块变异兽的骨头——那是他从那只C级巨兽身上取下来的，质地坚硬，可以打磨成武器或工具。在废土上，这种材料很值钱。

守卫的眼睛亮了。"进去吧。别惹事。"

楚锐点点头，走进聚集地。

里面的情况比他想象的要好。虽然还是很破旧，但至少比他在外面看到的那些废墟要强。有人在生火做饭，有人在修理工具，还有几个小孩在追逐打闹——这在废土上是很少见的景象。小孩通常活不长。

他走向一个看起来像是集市的地方。几个摊位摆在那里，卖着各种东西：罐头、破布、工具、还有一些他从没见过的东西。

"那个是什么？"他指着一个摊位上的一个小瓶子。瓶子里装着一种蓝色的液体，在光线下闪烁着诡异的光芒。

摊主是一个独眼的老头。"辐射血清。"他说，"可以暂时提高对辐射的抗性。你要吗？"

"多少钱？"

"你手里的那块骨头，换三瓶。"

楚锐摇摇头，继续往前走。他现在不需要那个。他需要信息。

然后他看到了一个摊位，上面摆着一些书。真正的书，纸质的，在这个数字设备都变成废铁的世界里，书是稀有的东西。

他走过去，蹲下，翻看着那些书。大部分是旧时代的垃圾——小说、杂志、还有一些他不认识的技术手册。但其中有一本吸引了他的注意。

那是一本手写的笔记本，封面已经磨损了，但上面的字迹还能看清。

"这是什么？"他问摊主。

摊主是一个中年女人，看起来很憔悴。"我丈夫的。"她说，"他是个...研究者。在降临日之前，他是科学家。"

"研究什么的？"

"外星人。"女人的声音变低了，"他研究那些外星人，研究他们的技术，研究他们的...目的。"

楚锐的心跳加速了。"我能看看吗？"

"可以。"女人说，"但价格不低。"

"我有这个。"楚锐又掏出一块兽骨，比刚才那块更大，"够吗？"

女人的眼睛亮了。"够了。还多了。"

她把笔记本递给他，同时给了他一个小布袋。"还有一些他的笔记，零散的。你也拿去。"

楚锐接过笔记本和布袋，走到一个角落，坐下来开始看。

笔记本的第一页写着：

"第9号生物质牧场观察记录。记录者：陈明。日期：降临日后第1年。"

楚锐开始读。

那些内容让他震惊。

外星人不是单一的种族。他们是...一个联盟，一个由几十个高等文明组成的收割者联盟。他们把宇宙中的星球分为不同的等级，然后按照等级进行"收割"。地球是第9号牧场，属于最低等级的——碳基生命牧场。

收割的周期是3000年。上一次收割是在3000年前，那时候人类还在石器时代。下一次收割...就是现在。

但笔记本里还有更重要的信息。关于"纯净脑域"的。

纯净脑域不是随机的。它是一种...进化，一种在极端压力下才会触发的基因突变。拥有纯净脑域的人，脑域开发度会自然超过7%，他们的灵魂能量比普通人类强十倍。

外星人需要这种能量。他们用这种能量来驱动他们的飞船，他们的武器，他们的...

楚锐的手在发抖。他想起了楚瑶，想起了她说的那些话——"有人在我的脑子里叫我"。她不是生病了，她是在...觉醒？

他继续往下读。然后，他看到了一段让他血液都凝固的话：

"纯净脑域的提取是致命的。一旦提取完成，宿主的灵魂会被完全抽干，身体变成空壳。没有例外。没有幸存者。"

楚锐把笔记本合上，深深地吸了一口气。

他感觉到一种...冰冷的愤怒。那种愤怒不是热的，不是燃烧的，是那种冻结一切的、冰冷的恨。

"引擎。"他在心里说，"你能感觉到她吗？"

【距离太远。】引擎说，【但如果你能进入那些采收舰，如果你能接近她的位置...也许可以。】

"怎么进去？"

【变强。强到足以撕裂那些飞船的护盾，强到足以在那些外星人的攻击下生存。】

"我需要多强？"

【铸基境。至少是铸基境。你现在只是凡星境，还差两个大境界。】

楚锐站起来，把笔记本收好。他看向天空，看向那些悬浮在轨道上的采收塔。

"等着我，楚瑶。"他说，"不管是死是活，我都会找到你。"

他转身，走出聚集地，再次走向废墟的深处。他需要更多的猎物，更多的力量，更多的...

他需要一个计划。
