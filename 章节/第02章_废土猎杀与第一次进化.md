# 第2章：废土猎杀与第一次进化

**降临日后第30天，14:15。**

**7号收割区，地表废墟。**

警报声像一把锯子，撕扯着7号收割区浑浊的空气。

楚锐站在提取室的金属走廊上，手中握着那把从扎戈尸体上缴获的高频震荡刃。刀身发出轻微的嗡鸣，那是高频振动模块运转的声音，像是一只被驯服的野兽在低吼。

他的视野中，真理引擎的界面无声地运转着。

【扫描范围：半径500米】
【检测到外星生命体：17个】
【D级（普通守卫）：12个】
【C级（低级军官）：4个】
【B级（区域指挥官）：1个】

【建议战术：】
【1. 优先猎杀D级目标，积累生物电能】
【2. 避免与B级目标正面接触】
【3. 利用废墟地形进行游击】

楚锐接受了这个建议。

他不是莽夫。三年的废土生存教会了他一个道理：活下来的才是赢家，死了的连复仇的资格都没有。

他深吸一口气，感受着体内流淌的力量。

【当前状态】
【等级：Tier 0 Level 1】
【生物电能：78%】
【已解锁能力：灵魂抽取（初级）、基础扫描】
【今日剩余使用次数：灵魂抽取 2/3】

灵魂抽取每天只能用三次。刚才对付扎戈和两个守卫，已经用掉了一次。剩下的两次，必须用在刀刃上。

楚锐握紧震荡刃，身形一闪，消失在走廊的拐角处。

---

**14:20。**

7号收割区的地表是一片废墟。

这里曾经是亚洲最大的城市群之一，有着摩天大楼、立交桥、地下商业街。现在只剩下扭曲的钢筋、破碎的玻璃、和无处不在的辐射尘。

风吹过废墟，发出呜咽般的声音，像是这座城市尚未死去的灵魂在哭泣。

楚锐贴着一面半倒塌的混凝土墙，呼吸放得极轻。他的眼睛透过墙上的裂缝，观察着前方五十米处的一个外星巡逻小队。

四个D级守卫。标准的巡逻配置。

它们的外形和扎戈有些相似，但体型更小，只有两米高，外骨骼也更单薄。它们手持的是D级等离子步枪，威力足以在混凝土墙上烧出一个洞，但射程和射速都很有限。

【目标：扎尔贡巡逻队（D级×4）】
【威胁等级：低】
【建议战术：利用废墟接近，近战优先，避免被集火】
【预计解决时间：45秒】

45秒。

楚锐计算着时间。巡逻队的换岗周期是10分钟，刚才的警报已经过去5分钟，再过5分钟就会有增援赶到。

他必须在5分钟内解决战斗，然后撤离。

楚锐深吸一口气，然后——

冲了出去。

不是直线冲刺，而是之字形的不规则移动。这是他在废土上练习了三年的移动方式，利用每一块碎石、每一根钢筋作为掩护，让敌人无法锁定他的轨迹。

"敌袭！"

巡逻队发现了他的踪迹。四个守卫同时抬起等离子步枪，蓝色的能量束划破空气，在楚锐身后的墙壁上烧出一个个焦黑的坑洞。

但楚锐太快了。

生物电能充盈的肌肉爆发出超越人类极限的力量，他的每一步都能跨出三四米远，身形在废墟间闪烁，像是一道黑色的闪电。

二十米。

十米。

五米。

第一个守卫终于锁定了他的轨迹，等离子步枪的枪口喷吐出致命的光芒。但就在能量束即将命中楚锐的瞬间，他的身形诡异地一侧——

【基础扫描】发动。

在他的视野中，那道能量束的轨迹被清晰地标记出来，就像是慢动作回放一样。他提前0.3秒预判了射击角度，身体自然而然地做出了规避动作。

能量束擦着他的肩膀飞过，高温烧焦了他的衣服，但没有伤到他分毫。

然后，楚锐出现在了第一个守卫的面前。

守卫的复眼中闪过一丝惊恐。它想要后退，想要重新瞄准，但已经太迟了。

楚锐的右手如闪电般探出，五指并拢如刀，狠狠地刺入了守卫胸口的甲壳缝隙——那里是基础扫描标记出来的弱点，神经中枢的节点。

【灵魂抽取】

发动。

守卫的身体瞬间僵直。它甚至来不及发出惨叫，生命的本质就被那只人类之手硬生生地拽了出来。

那是一团比扎戈更小、更暗淡的蓝色光团。楚锐没有犹豫，直接将它吸入了体内。

【灵魂抽取成功】
【获得灵魂能量：微量】
【生物电能恢复15%】

与此同时，另外三个守卫终于反应过来，等离子步枪的枪口同时对准了楚锐。

"去死吧，猴子！"

三道光束同时射出，封锁了楚锐所有的闪避角度。

但楚锐没有闪避。

他左手一扬，那把从扎戈身上缴获的高频震荡刃脱手而出，在空中划出一道银色的弧线——

【目标：D级守卫×3】
【弱点：颈部甲壳缝隙】
【距离：7.3米】
【建议投掷角度：23度】

真理引擎的计算精确到毫秒。震荡刃在空中旋转了半圈，准确地切入了第一个守卫的颈部，然后余势不减地贯穿了第二个守卫的胸口，最后卡在了第三个守卫的肩胛骨上。

高频振动的刀刃轻易地撕裂了D级守卫单薄的外骨骼，就像切开豆腐一样轻松。

三个守卫同时倒地。

楚锐走上前，从尸体上拔出了震荡刃，然后依次将手按在它们的胸口。

【灵魂抽取】×3

【灵魂抽取成功×3】
【获得灵魂能量：微量×3】
【生物电能恢复35%】
【当前生物电能：100%（已满）】

【检测到生物电能溢出……】
【是否进行等级突破？】

楚锐一愣。

等级突破？

【说明：生物电能已达到当前等级上限，可选择：】
【A. 将溢出能量转化为身体强化（永久提升基础属性）】
【B. 尝试突破至 Tier 0 Level 2（有15%失败概率，失败后进入24小时虚弱期）】

楚锐没有犹豫。

"突破。"

【等级突破开始……】

剧痛。

比之前任何一次都要剧烈的疼痛瞬间席卷了他的全身。他感觉自己的每一个细胞都在被撕裂、重组、再撕裂、再重组。基因链在高温下扭曲、断裂、重新连接。

他跪倒在地，双手死死地抓住地面，指甲在混凝土上划出深深的痕迹。

汗水如雨般从他的额头滴落，混杂着从毛孔中渗出的黑色杂质——那是他体内积累的毒素和辐射尘，被突破时的能量净化排出。

时间仿佛被拉长了。

一秒钟像一个世纪那么漫长。

楚锐咬紧牙关，不让自己发出惨叫。他知道，如果现在发出声音，就会把更多的敌人引过来。他必须忍耐。

终于——

【等级突破成功！】
【当前等级：Tier 0 Level 2】
【解锁能力：肉体强化（初级）】
【肉体强化：可局部硬化肌肉，提升防御力和力量】
【基础属性提升：力量+30%，速度+20%，恢复力+50%】

楚锐缓缓站起身，感受着体内涌动的全新力量。

他握了握拳，空气中传来一声清脆的音爆。

这是……质变。

【当前状态】
【等级：Tier 0 Level 2】
【生物电能：23%】
【已解锁能力：灵魂抽取（初级）、基础扫描、肉体强化（初级）】
【今日剩余使用次数：灵魂抽取 0/3（已用完）】

灵魂抽取的次数用完了。但在突破后，他的基础属性大幅提升，即使不用能力，正面战斗力也足以碾压D级守卫。

更重要的是，他解锁了新的能力——【肉体强化】。

楚锐心念一动，发动了【肉体强化】。

他的右臂肌肉瞬间膨胀，皮肤变得如同钢铁般坚硬，泛着淡淡的金属光泽。他能感觉到，这一拳下去，足以打穿混凝土墙壁。

"有趣。"

一个声音突然从废墟的阴影中传来。

楚锐猛地转身，高频震荡刃横在胸前，全身肌肉绷紧，进入了战斗状态。

"谁？"

阴影中，一个人影缓缓走出。

那是一个女人。看起来二十岁左右，身材修长，穿着一件由各种废料拼凑而成的灰色斗篷。她的脸被兜帽遮住了一半，只露出下巴和一张苍白的嘴唇。

但最引人注目的，是她的眼睛。

那是一双深紫色的眼睛，瞳孔中似乎有星光在流转，美丽而神秘。

"别紧张。"女人的声音很轻柔，带着一种奇异的韵律，"如果我想对你不利，刚才你突破的时候，就已经死了。"

楚锐没有放松警惕。真理引擎正在疯狂地扫描着眼前的女人。

【目标：未知人类女性】
【威胁等级：无法评估】
【特殊状态：检测到高强度脑域活动，疑似精神力觉醒者】
【建议：保持警惕，不要直视其双眼】

精神力觉醒者？

楚锐皱起眉头。他在废土上听说过这种存在——那些天生脑域开发度就远超常人的"异能者"。他们不需要科技，不需要改造，只需要依靠纯粹的精神力，就能做到各种不可思议的事情。

但这种人极其罕见。据说一万个人里才有一个，而且大部分都在降临日的时候被外星人抓走做实验了。

"你是谁？"楚锐冷声问道。

女人微微一笑，那笑容中带着一丝苦涩。

"我叫苏星陨。"她说，"和你一样，也是这屠宰场里的……猎物。"

"只不过，我一直在等待一个能够反杀猎手的人出现。"

她抬起头，深紫色的眼眸直视着楚锐，目光中带着一种近乎偏执的期待。

"而你，楚锐，我等的就是你。"
