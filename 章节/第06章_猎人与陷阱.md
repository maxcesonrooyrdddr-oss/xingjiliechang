# 第6章：猎人与陷阱

**降临日后第35天。**

**7号收割区，废弃工厂。**

胜利的喜悦是短暂的。

当楚锐带领星火小队返回04号掩体时，等待他们的是林珩凝重的面容。

"我们被盯上了。"林珩直截了当地说，"刚才的伏击，引来了更大的鱼。"

他指着地图上的一个位置，那里标注着一个醒目的红点。

"B级指挥官，奥莱克。"

"它是7号收割区的实际掌控者，仅次于总督的存在。它的实力，至少是扎戈的十倍以上。"

楚锐皱起了眉头。

【目标：奥莱克（扎尔贡高级指挥官）】
【威胁等级：B级】
【预估战力：相当于Tier 0 Level 9或Tier 1 Level 3】
【当前状态：正在朝04号掩体方向移动，预计6小时后到达】

【警告：以宿主当前实力，正面交锋胜率低于5%】

5%。

几乎必死的局面。

"我们撤。"楚锐当机立断，"所有人，带上能带的东西，立刻撤离04号掩体。"

"往哪里撤？"有人问道。

楚锐看向地图，目光落在了一个位置上。

"太平洋底。"

"外星人在那里建造了一座行星发动机，意图将地球拖走作为矿星。那里防御严密，但……也是它们最想不到我们会去的地方。"

"最危险的地方，就是最安全的地方。"

林珩点了点头。

"好。我带队掩护，你带主力突围。"

"不。"楚锐摇了摇头，"你带主力突围，我带精锐小队断后。"

"楚锐——"

"这是命令。"楚锐打断了他，声音中带着一种不容置疑的威严，"我是星火的领袖，我有责任保护我的同伴。"

"而你，林珩，你是人类的统帅，你必须活下去。"

林珩沉默了。

他看着眼前这个年轻人，看着他那双燃烧着火焰的眼睛，突然有一种感觉——

这个人，真的能够带领人类走出黑暗。

"好。"他说，"但你必须答应我一件事。"

"什么事？"

"活着回来。"

楚锐微微一笑，那笑容中带着一种让人心安的力量。

"放心。"他说，"我还没杀够呢。"

---

**6小时后。**

**04号掩体外围，废墟地带。**

楚锐带着一支由30人组成的精锐小队，埋伏在废墟的阴影中。

他们每个人都穿着由塔克特制的"废土钢"轻型装甲，手持改良版震荡刃，腰间挂满了辐射手雷。

这是他们全部的家底，也是他们最后的希望。

"来了。"苏星陨的声音在楚锐脑海中响起，"东南方向，距离3公里，数量……50人以上。"

"其中有4个C级，1个B级。"

楚锐深吸一口气，握紧了手中的震荡刃。

【目标：奥莱克率领的追击部队】
【兵力：D级×45，C级×4，B级×1】
【战术建议：利用废墟地形进行游击，优先消耗D级兵力，避免与B级正面接触】

【警告：B级指挥官拥有区域性精神压制能力，请小心】

精神压制？

楚锐转头看向苏星陨。

"你能对抗它的精神压制吗？"

苏星陨深紫色的眼眸中闪过一丝凝重。

"可以试试。"她说，"但只能维持很短的时间。"

"足够了。"楚锐说，"你只需要给我争取30秒。"

"30秒，我就能杀到它面前。"

苏星陨点了点头，闭上了眼睛。

她的精神力开始扩散，在废墟上空编织成一张无形的大网，等待着猎物的到来。

---

十分钟后，外星部队进入了伏击圈。

为首的，是一个比扎戈还要高大魁梧的身影。它的外骨骼呈现出一种暗金色的光泽，每一步踏出，都会在地面上留下一个深深的脚印。

这就是B级指挥官，奥莱克。

"停。"奥莱克突然抬起手，示意部队停止前进。

它六只复眼扫视着周围的废墟，似乎察觉到了什么。

"有埋伏。"它冷冷地说道，"出来吧，虫子们。我知道你们在那里。"

没有回应。

只有风吹过废墟的呜咽声。

"哼，不见棺材不落泪。"奥莱克冷笑一声，然后——

【精神压制】发动。

一股无形的精神波动瞬间扩散开来，笼罩了整个废墟区域。

埋伏在阴影中的星火战士们顿时感觉脑袋像是被重锤击中，视野开始模糊，四肢变得沉重。

"就是现在！"楚锐大吼。

苏星陨猛然睁开眼睛，深紫色的眼眸中爆发出耀眼的光芒。

【心灵屏障】全开！

两股精神力在空中碰撞，发出一声只有精神力者才能听见的巨响。

苏星陨闷哼一声，嘴角溢出一丝鲜血，但她咬紧牙关，死死地支撑着那道屏障。

"楚锐，快！"

楚锐没有犹豫。

他的身体如同离弦之箭般射出，在肉体强化的加持下，他的速度快到了极致，甚至在空气中留下了一道淡淡的残影。

【灵魂抽取】发动！

目标是……奥莱克身边的四个C级护卫！

在苏星陨的心灵屏障保护下，奥莱克的精神压制被暂时抵消，那四个C级护卫完全没有反应过来，就被楚锐一一近身，灵魂被强行抽离。

【灵魂抽取成功×4】
【获得灵魂能量：中等×4】
【生物电能恢复至100%】
【检测到生物电能溢出……】
【是否尝试突破？】

"突破！"

【等级突破开始……】

剧痛再次席卷全身，但楚锐咬紧牙关，强忍着痛苦，继续冲向奥莱克。

他的等级在战斗中突破，从Tier 0 Level 3直接跃升到了……

【突破成功！】
【当前等级：Tier 0 Level 5】
【解锁能力：动能感知（初级）——可预判敌方攻击轨迹】

奥莱克终于反应了过来。

它发出一声愤怒的咆哮，暗金色的外骨骼上亮起了密密麻麻的能量纹路，一拳朝着楚锐轰来。

那一拳的力量，足以粉碎一辆坦克。

但楚锐已经预判到了。

【动能感知】发动！

在他的视野中，奥莱克拳头的轨迹被清晰地标记出来，就像是一条红色的箭头。

他提前0.5秒做出了规避动作，身体以一种不可思议的角度扭曲，险之又险地避开了那一拳。

然后，他的震荡刃，刺向了奥莱克的胸口。

铛！

刀刃与暗金色外骨骼碰撞，发出一声刺耳的金属摩擦声。

没有刺进去。

B级的外骨骼，硬度远超C级。

"愚蠢。"奥莱克冷笑，"就凭这种武器，也想伤到我？"

它另一只手如闪电般探出，死死地抓住了楚锐的脖子，将他提了起来。

"你以为杀了我的几个手下，就能挑战我？"

"愚蠢的虫子，我会让你知道，什么叫做……绝望。"

它开始收紧手指。

楚锐感觉自己的颈骨在发出不堪重负的呻吟，视野开始发黑。

但他没有恐惧。

他的嘴角，反而勾起了一丝笑容。

"你上当了。"

奥莱克一愣。

下一秒，它感觉到了。

在它的脚下，废墟的深处，传来了一阵剧烈的震动。

那是……

"引爆！"楚锐用尽最后的力气大吼。

轰！！！

预先埋在废墟下的高浓度辐射废料炸弹，同时爆炸了。

那是楚锐用【废料重构】能力制造的临时炸弹，威力足以摧毁一座大楼。

奥莱克被爆炸的冲击波吞没了。

而楚锐，在爆炸的前一刻，被一道紫色的光芒包裹，瞬间传送到了百米之外。

那是苏星陨用尽全部精神力发动的【紧急传送】。

她倒在地上，脸色苍白如纸，已经失去了意识。

但她的嘴角，却带着一丝满足的微笑。

因为他们赢了。

废墟的烟尘中，奥莱克那暗金色的身影摇摇欲坠。

它的外骨骼被炸得支离破碎，绿色的血液从伤口中不断涌出。

它还没有死，但已经重伤。

楚锐捡起掉在地上的震荡刃，一步一步走向它。

"你……"奥莱克看着他，复眼中第一次出现了恐惧，"你到底是谁？"

楚锐站在它面前，高举着手中的刀。

"我是楚锐。"他说，"星火反抗军的领袖。"

"也是……你们的噩梦。"

刀光落下。

【灵魂抽取】！

这一次，是一团前所未有的巨大蓝色光团，那是B级强者的灵魂，蕴含着惊人的能量。

楚锐将它吞噬，感受着体内力量的暴涨。

【灵魂抽取成功】
【获得灵魂能量：大量】
【获得记忆碎片：7号收割区完整布防图、总督府内部结构、行星发动机核心位置】
【等级提升：Tier 0 Level 5 → Tier 0 Level 7】

【当前等级：Tier 0 Level 7】

距离Tier 1，只有一步之遥。

楚锐抱起昏迷的苏星陨，转身看向身后幸存的星火战士们。

"我们走。"他说，"去太平洋，去行星发动机。"

"在那里，我们将点燃……燎原之火。"

**第6章：猎人与陷阱 · 完**
