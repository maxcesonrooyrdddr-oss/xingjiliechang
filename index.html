📤 手动更新指南

  1. 在左侧列表中点击想要编辑的章节
  2. 点击"编辑"按钮进入编辑模式
  3. 修改内容后，点击"下载文件"保存到电脑
  4. 访问 GitHub 仓库：
    https://github.com/maxcesonrooyrdddr-oss/xingjiliechang
  5. 进入 章节/ 文件夹
  6. 点击 "Add file" → "Upload files"
  7. 上传下载的 .md 文件（同名替换）
  8. 点击 "Commit changes" 确认
  9. 返回本页面，点击顶部 🔄 刷新查看
